通视分析（Line of Sight）

使用：

可视域分析（Viewshed）参数补充

  • 输入参数建议：
    • viewPosition: [lng, lat, alt] 观察点
    • targetPositions: Array<[lng, lat, alt]> 目标点集（或以水平/垂直角构造扇形范围）
    • maxDistance: number 最大可视距离（米）
    • verticalAngle: number 垂直视角范围（度）
    • horizontalAngle: number 水平视角范围（度）
  • 输出结果：
    • 每个目标的可视/不可视标记、原因、距离；并可生成一个扇形体或网格标注

绘制锥体时注意：

  • 使用 PolylineVolume 或自定义 Primitive 生成锥体边界；
  • 大量网格时建议批量合并（Primitive 集合），避免 Entity 爆炸。

剖面采样与坡度/坡向（细节）

  • 采样策略：
    • 将起终点插值为 N 段（例如 256）；
    • 使用 sampleTerrainMostDetailed（若使用 Cesium 原生地形采样）或 Mars3D 封装进行批量采样；
    • 结果保存为 { distance, elevation, position } 数组。
  • 坡度（slope）计算：
    • 对相邻采样点 (i,i+1)，计算水平距离与高差，slope% = (Δh / Δd) * 100；
    • 坡角 angle = atan(slope/100)（转度）。
  • 坡向（aspect）计算：
    • 对每个采样段，取水平向量方位角（基于经纬投影近似或在 ECEF 上投影后求方位角）。

淹没分析（参数与单位一致性）

  • 建议属性：
    • bounds: 多边形区域（[lng,lat] 顶点数组）
    • baseHeight: 地形基准高程（米）或从地形采样自动估计
    • waterLevel: 当前水位（米，高程基准同 baseHeight）
    • step: 计算步长（米），用于体积网格化估算
  • 体积估算：
    • 将区域划分网格，网格中心采样地形高程 h(x,y)；
    • 若 waterLevel > h，单元体积约 Δx * Δy * (waterLevel - h)；
    • 总体积为各单元体积之和。

注意：统一所有高度的参考基准（同一椭球、同一地形数据源），避免相对/绝对高混用。

阴影/日照分析（简述）

  • 启用阴影：
    • viewer.shadows = true；
    • 配合设定 light（太阳）时间 viewer.clock.currentTime 控制日照角度。
  • 动态时间序列：
    • 在日间对多个时间采样，统计对象被照/被遮时间比例，形成日照分析结果。
